# 사립유치원 현금출납부 목별 검토 도구 v6

현금출납부 Excel 파일을 브라우저에서 읽어, 2026 사립유치원 예·결산 및 적립금 업무처리 지침 기준으로 목 부적정 의심·추가 확인 항목을 선별하는 무료 정적 웹사이트입니다.

## v6 반영 내용

- 예·결산 지침 업로드 제거: 현금출납부만 업로드하여 검토
- 결과 보기 탭 추가: 전체 / 수입 / 지출 / 목 부적정 의심
- `[과오납반환결의]`, `과오납반환`, `과오납환불` 건은 검토 대상에서 제외하고 결과에도 표시하지 않음
- `[과목경정]`, `과목경정`, `목경정` 행을 인식하여 동일 제목·동일 금액의 정상 비목 경정이 확인되면 기존 지적 후보에서 제외
- 수입 금액 열 선택 개선: `납입금명`의 `3월` 같은 텍스트를 금액으로 오인하지 않도록 수정
- 법정부담금/퇴직금 및 퇴직적립금 정상 처리 보정
- `보수`, `수리`, `구입`, `재료`, `교육비` 관련 오탐 보정

## 중요한 한계

이 도구는 제목·적요 기반 1차 선별 도구입니다. 최종 지적 여부는 품의서, 지출결의서, 계약서, 거래명세서, 검수조서 등 증빙 확인 후 판단해야 합니다.


## v9 변경사항

- 단일 정답 비목 단정 방식을 완화하고, 일부 항목은 허용 비목 예외 규칙으로 처리합니다.
- 방과후 급식·간식, 교직원 급식비, 교직원공제회비, 법정부담금, 퇴직적립금 관련 오탐을 줄였습니다.
- 결과는 확정 지적이 아니라 증빙 확인을 위한 1차 선별 자료입니다.
